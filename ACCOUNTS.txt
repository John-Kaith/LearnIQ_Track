================================================================================
                            HOW TO LOGIN
================================================================================

1. Open frontend/login.html in your browser
2. Enter ID Number (from list above)
3. Enter Email (from list above)
4. Enter Password (from list above)
5. Click Login

Note: Role field has been removed from login form.
      The system automatically determines user's dashboard based on their role.

Admin → admin-approval.html
Teacher → teacher-dashboard.html
Student → student-dashboard.html

                         NOTES
================================================================================

- Sample data is used for demonstration purposes
- No backend server required for frontend testing
- All user data resets on localStorage clear
- Status badges indicate account approval state:
  * Pending = Awaiting admin approval
  * Approved = Can login successfully
  * Rejected = Login attempt blocked
